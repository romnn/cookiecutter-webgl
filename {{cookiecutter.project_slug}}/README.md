#### Setup
1. Install dependencies
    ```bash
    yarn
    ```

2. Run project in watch mode
    ```bash
    yarn start
    ```
    **Notice:** Remember to run `yarn lint`, especially before you commit any changes.

3. Build project
    ```bash
    yarn build
    ```
